# Habit_planner

This is my second program that I created and finished in January-February 2026.
It is a simple habit planner that helps the user plan and follow along his/her life's important habitual activities.

This program includes options to:
+ view one's daily habit progress
+ delete and add habits to one's list
+ view habit progression throughout a longer period with streaks and day count of unfinished habits
+ mark habits as complete

There are also functionalities to:
+ warn the user when a habit is unfinished and it's already later in the evening;
+ track habit completion streaks, and to congratulate a user for each week of completion;
+ automatically reset progress based on last login.

Additionally, this program incorporates separation of concerns as it has separate files for logic, data persistence, and user interaction.
May this Habit Planner serve you well.
